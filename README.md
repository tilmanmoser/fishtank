# Fish Tank (Flocking Boids)

Simulation of [flocking boids algorithm](https://en.wikipedia.org/wiki/Boids) within a fish tank made with pygame.

# Installation

```
echo "PIPENV_VENV_IN_PROJECT=1" > .env
pipenv install
```

# Run

```
pipenv run python main.py
```
